Developed Correspondent Bank API Services at Bank BRI to assess Money Laundering (ML) and Terrorism Financing (TPPT) risks associated with Correspondent Banks. Created an application for objective risk measurement, utilizing MinIO for file storage, MySQL as the main database, and Golang for API services.
